쿼리문은 데이터베이스에서 데이터를 조회하거나 삽입, 업데이트. 삭제하는데 사용되는 명령어의 집합
주로 SQL(Structured Query Language)을 사용하여 데이터베이스와 상호작용

SELECT
- SELECT 문은 데이터베이스에서 데이터를 조회할 때 사용
- 기본 형식은 'SELECT [칼럼명] FROM [테이블명] WHERE [조건식]'
- '[칼럼명]'에는 조회하고자 하는 열 이름을 나열
- '[테이블명]' 은 데이터를 조회할 테이블 이름
- '[조건식]'은 서택 사항, 특정 조건을 지정하여 원하는 데이터만 선택할 수 있음

INSERT INTO
- INSERT INTO 문은 데이터베이스에 새로운 레코드를 삽입할 때 사용
- 기본 형식은 'INSERT INTO [테이블명] ([칼럼1],[칼럼2],...) VALUES ([값1], [값2], ...)'
- '[테이블명]'은 새로운 레코드를 삽입할 테이블 이름
- '[칼럼1], [칼럼2], ...' 은 레코드에 삽입할 열 이름들을 나열
- '[값1], [값2], ...' 은 '[칼럼1],[칼럼2], ...'에 대응하는 값을 나열

UPDATE
- UPDATE 문은 데이터베이스의 기존 레코드를 업데이트할 때 사용
- 기본 형식은 'UPDATE [테이블명] SET [칼럼1] = [값1], [칼럼2] = [값2], ... WHERE [조건식]'
- '[테이블명]'은 업데이트할 테이블의 이름
- '[칼럼1], [칼럼2], ...' 은 업데이트할 열 이름들 나열
- '[값1], [값2], ...' 은 '[칼럼1], [칼럼2], ...'에 대응하는 새로운 값 나열
- '[조건식]' 은 선택 사항, 업데이트를 수행할 레코드를 제한할 수 있는 조건 지정

DELETE
- DELETE 문은 데이터베이스에서 레코드를 삭제할 때 사용
- 기본 형식은 'DELETE FROM [테이블명] WHERE [조건식]'
- '[테이블명]' 은 삭제할 레코드를 포함하는 테이블의 이름
- '[조건식]' 은 선택 사항, 삭제할 레코드를 제한하는 조건을 지정, 조건을 생략하면 모든 레코드가 삭제될 수 있음

기타
- JOIN을 사용하여 여러 테이블을 연결
- GROUP BY를 사용하여 그룹별로 데이터를 집계할 수 있음

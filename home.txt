홈 화면 기능 구현

1. 홈 화면 디자인: Flask는 템플릿 엔진을 지원, jinja2를 사용하여 동적인 HTML 템플릿 생성 가능
홈 화면의 디자인을 구성하고 필요한 정보를 템플릿에 전달하여 동적으로 렌더링할 수 있음.
render_template 함수 사용하여 템플릿을 렌더링, 필요한 데이터를 템플릿에 전달 가능

2. 홈 화면 기능: 필요한 데이터를 데이터베이스에서 가져와 화면에 표시,
데이터베이스에 쿼리를 실행하여 필요한 데이터를 가져오고, 이를 템플릿에 전달하여 렌더링 가능

3. 인증된 사용자: 홈 화면에 접근하기 위해서는 인증된 사용자만 접근할 수 있도록 보호할 수 있음
Flask_JWT_Extended를 사용하여 인증과 접근 제어 구현
@jwt_required 데코레이터를 사용하여 특정 뷰 함수에 접근 제한 설정 가능
get_jwt_identity 함수를 사용하여 현재 인증된 사용자의 식별 정보를 가져올 수 있음

4. 기타 기능: 데이터 받아 처리, 데이터베이스에 저장하거나 업데이트 가능


------------------------------------- 새로운 시도 --------------------------------------------

"Missing Authorization Header" 오류

인증 헤더가 요청에 누락되었거나 잘못된 형식으로 제공되었을 때 발생

JWT 토큰은 Authorization 헤더에 포함되어야 함

Authorization 헤더 값은 Bearer<토큰값> 형식이어야 함

Authorization(키) - Bearer <토큰값>
